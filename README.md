\# Kyklos



Private development repository for orbital mechanics Python package.



\## Features



\- High-performance trajectory propagation using Heyoka integrator

\- Support for 2-body, 3-body (CR3BP), and perturbed dynamics

\- Coordinate transformations (Keplerian, Cartesian, Equinoctial)

\- State estimation and formation flying applications



\## Setup

```bash

\# Clone repository

git clone https://github.com/shbi6911/kyklos\_private.git

cd kyklos\_private



\# Create conda environment

conda create -n kyklos python=3.11

conda activate kyklos



\# Install dependencies

pip install -e ".\[dev]"

```
